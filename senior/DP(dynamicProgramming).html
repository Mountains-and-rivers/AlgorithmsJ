动态规划： 从底层开始解决问题，将所有的小问题解决掉，然后合并成一个整体解决方案
     
        简单版本的动态规划，
            计算斐波拉切数列：
          function recurFib(){
              if(n<2) {
                 return n;
              }else {
                 return recurFib(n-1) + recurFib(n-2);
              }
          }
        console.log(recurFib(10));
